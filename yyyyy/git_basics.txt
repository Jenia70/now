Основы использования Git

1. Создание нового репозитория:
   - `git init` - инициализация нового репозитория.

2. Создание новой папки и файла:
   - `mkdir my_project` - создание новой папки.
   - `cd my_project` - переход в созданную папку.
   - `touch myfile.txt` - создание нового файла.

3. Добавление и коммит изменений:
   - `git add myfile.txt` - добавление файла в индекс.
   - `git commit -m "Initial commit"` - создание коммита с сообщением.

4. Подключение к удаленному репозиторию на GitHub:
   - Создать новый репозиторий на GitHub.
   - `git remote add origin <repository_url>` - связать локальный репозиторий с удаленным.

5. Отправка изменений на GitHub:
   - `git push -u origin master` - отправка изменений на GitHub.

Это базовые шаги для работы с Git. Можно использовать эти команды для создания, отслеживания и совместной работы над проектами.
